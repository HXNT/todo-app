# To-Do List - Свой список задач

Адаптивное веб-приложение для планирования и управления ежедневными задачами.

Реализовано с помощью HTML, CSS, JavaScript.

## Функционал

### Базовый функционал:
* **Добавление задач:** Добавление задачи через поле ввода (с проверкой на пустую строку).
* **Отображение списка:** Вывод актуального списка задач.
* **Отметка о выполнении:** Возможность переключать статус задачи (при выполнении задачи текст зачеркивается, сама задача становится тусклее по цвету, слева стоит галочка), а также изменение в отображении счетчика Завершенных или Активных задач внизу списка.
* **Удаление задач:** Мгновенное удаление выбранной задачи из списка при нажатии на крестик, либо удаление всех завершенных задач по кнопке снизу.

### Дополнительные функции:
* **Сохранение в LocalStorage:** Все данные сохраняются в браузере. При обновлении страницы или закрытия вкладки задачи не исчезают.
* **Фильтрация задач:** Удобная навигация по вкладкам: *Все* / *Активные* / *Завершенные*.
* **Адаптивная верстка:** Интерфейс оптимизирован под любые устройства - приложение выглядит одинаково как на ПК, так и на смартфоне.
* **Кастомный дизайн:** Реализован интерфейс с темной темой.

## Как запустить

### Локально:

1. Перейдите в папку, в которой будет храниться проект.

```bash
cd C:\Ваш_Путь
```

2. Загрузите проект.

```bash
git clone https://github.com/HXNT/todo-app.git
```

3. Перейдите в папку проекта.

```bash
cd C:\Ваш_путь\todo-app
```

4. Запустите проект.

```bash
start index.html
```

Либо можно запустить с помощью **Github Pages**:

https://hxnt.github.io/todo-app/
