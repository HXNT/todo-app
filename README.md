# TODO — Веб-приложение «Список задач»

Простое и стильное веб-приложение для управления задачами, реализованное на **HTML + CSS + JS** (фронтенд) и **Python Flask** (бэкенд).

---

## Стек технологий

| Слой       | Технология          |
|------------|---------------------|
| Фронтенд   | HTML5, CSS3, Vanilla JS |
| Бэкенд     | Python 3, Flask     |
| Хранилище  | JSON-файл (`tasks.json`) |
| Стили      | Google Fonts (Playfair Display, DM Sans) |

---

## Структура проекта

```
todo-app/
├── app.py                  # Flask-сервер, REST API
├── requirements.txt        # Python-зависимости
├── tasks.json              # Хранилище задач (создаётся автоматически)
├── templates/
│   └── index.html          # HTML-шаблон приложения
└── static/
    ├── css/
    │   └── style.css       # Стили, анимации, адаптивная вёрстка
    └── js/
        └── script.js       # Логика фронтенда, обращения к API
```

## Запуск проекта

### 1. Клонировать / скачать проект

```bash
git clone <your-repo-url>
cd todo-app
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Запустить сервер

```bash
python app.py
```

### 4. Открыть в браузере

```
http://localhost:5000
```

## REST API

| Метод  | URL                       | Описание                        |
|--------|---------------------------|---------------------------------|
| GET    | `/api/tasks`              | Получить все задачи             |
| GET    | `/api/tasks?filter=active`| Только активные задачи          |
| GET    | `/api/tasks?filter=done`  | Только завершённые задачи       |
| POST   | `/api/tasks`              | Добавить задачу `{"title":"..."}` |
| PATCH  | `/api/tasks/<id>`         | Переключить статус задачи       |
| DELETE | `/api/tasks/<id>`         | Удалить задачу                  |
| DELETE | `/api/tasks/clear-done`   | Удалить все завершённые задачи  |

- ✅ Адаптивная вёрстка (мобильные устройства)
